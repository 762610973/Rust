# RUST学习🎉️ 

- 程序文件后缀名：rs
- 命名规范：采用下划线的形式
- 编译：rustc xxx.rs(适合小文件)
- Rust的缩进是4个空格而不是table
- `cargo version`
- `cargo new fileName`
- `cargo init`
- Rust的包称为`crate`
- 源代码都应该放在src目录下
- `cargo build --release`//构建可执行文件
- `cargo run`  //构建和运行文件
- `cargo check`//检查代码，确保能通过编译，但是不产生任何可执行文件，速度快·
- 尽量不要声明未使用的变量，向Go靠拢
